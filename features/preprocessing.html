Preprocessing¶

Descriptors¶

The input data for MALA are, in theory, the atomic positions. As MALA oeprates “per-grid-point” manner, information from the atomic positions have to be present on the entire grid of the simulation cell. This is done by calculating descriptors on the grid. Currently, only SNAP descriptors are supported. MALA uses LAMMPS to calculate these SNAP descriptors.

Targets¶

MALA is optimized for the usage of the LDOS (local density of states) as target quantity. The LDOS gives the DOS (density of states) at each grid point, and thus gives information on the energy-grid as well as the 3D grid. The LDOS can be used to efficiently calculate quantities of interest. MALA provides parsing routines to read the LDOS from DFT calculations.
